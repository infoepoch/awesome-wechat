# 微信支付相关
具体开发相关的文档在首页都已经给出，请详细阅读。

## 接入微信支付
- 公众号支付 在微信内的商家页面上完成公众号支付
- APP支付 在APP中，调起微信进行APP支付
- 扫码支付 扫描二维码(包含PC网站)进行扫码支付
- 刷卡支付 用户展示条码，商户扫描完成刷卡支付
- 微信买单 自助开通、无需开发，一分钟上线微信支付
